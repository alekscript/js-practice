Switch конструкция

Конструкция switch заменяет собой сразу несколько if. Она представляет собой более наглядный способ сравнить выражение сразу с несколькими вариантами. Конструкция switch имеет один или более блок case и необязательный блок default. let arg = prompt("Введите число?"); switch (arg) { case '0': case '1': alert( 'Один или ноль' ); break; case '2': alert( 'Два' ); break; case 3: alert( 'Никогда не выполнится!' ); break; default: alert( 'Неизвестное значение' ); }
